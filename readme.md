# Python Calculator

A simple console-based calculator that performs addition, subtraction, multiplication, and division operations. Built using Visual Studio Code with Git version control for a software development assignment.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Input validation and error handling (division by zero, invalid inputs)
- Continuous operation until user quits
